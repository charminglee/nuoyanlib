=======
API文档
=======

「nuoyanlib」包含以下公开的工具包：

- ``nuoyanlib.client``：客户端工具包
- ``nuoyanlib.server``：服务端工具包
- ``nuoyanlib.utils``：通用工具包
- ``nuoyanlib.extensions``：「nuoyanlib」扩展功能

其他以下划线开头的模块/函数/变量/类是私有的，用于内部实现，不保证功能稳定性。

.. warning::

    再次提醒，你只能在客户端/服务端环境导入 ``nuoyanlib.client`` / ``nuoyanlib.server`` 。
