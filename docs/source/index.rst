=================
「nuoyanlib」文档
=================

「nuoyanlib」是基于网易我的世界（我的世界中国版） `ModSDK <https://mc.163.com/dev/index.html>`_ 的开发工具库，封装了许多常用的客户端/服务端工具和通用算法，帮助开发者更高效地实现复杂功能。您无需对现有的代码结构作任何修改，即可轻松引入「nuoyanlib」。

| ✅ 已支持 **ModSDK 3.5**
| 📌 适合 **个人** 或 **团队** 项目，可自由用于 **商业** 和 **非商业** 用途
